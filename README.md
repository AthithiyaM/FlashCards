# CSV Flashcards

A simple, self-contained web app that loads a 2-column CSV and lets you flip through flashcards.

## Run

Open `index.html` in any modern browser. No build tools needed.

## CSV format

Two columns per row: `front,back`.

Example:

```
Capital of France,Paris
"2 + 2",4
"Term with, comma","Definition with, comma"
```

## Features

- Click the card (or press Enter/Space) to flip.
- Prev/Next navigation with wrap-around.
- Shuffle and reset order.
